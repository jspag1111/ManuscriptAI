# ChatKit Python API map

Source: `https://openai.github.io/chatkit-python/api/chatkit/`

Use this as a quick locator for where functionality lives.

- `chatkit.server.ChatKitServer`: core server wrapper for ChatKit endpoints.
- `chatkit.store.ChatKitStore`: storage interface for threads and items.
- `chatkit.types`: `ThreadMetadata`, `ThreadItem`, and related types.
- `chatkit.agents`: helpers for integrating the OpenAI Agents SDK with ChatKit.
